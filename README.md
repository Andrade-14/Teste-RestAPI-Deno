# Node.js REST API

Este é um projeto básico de API REST em Node.js.

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/Andrade-14/nodejs-rest-api.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Execute o servidor:
   ```bash
   npm start
   ```
